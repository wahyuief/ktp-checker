# KTP Checker
Identifikasi pola NIK KTP menggunakan golang akan menghasilkan output kecamatan, kota, provinsi, jenis kelamin dan tanggal lahir.

## How to Build
```
go build
```

## How to Run
```
./ktp-checker
```
